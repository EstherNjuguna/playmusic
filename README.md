## installation and Setup
clone this repo . You need node and npm installed
installation:
`npm install`
To Run Test Suite:
`npm test`
To start Server
`npm start`
The API used is hosted from AudioDb : https://theaudiodb.com/api/v1/json/2/artist.php?i=111522
